# CalculatorInBatch
Simple calculator for batch/cmd.

version 0.004812

Help:
 [*] - multiply
 [+] - add
 [-] - subtract
 [/] - divide

 Not work odd numbers, even number are selected automatically.
 When writing any character other than number it will display null/0
